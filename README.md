# Описание проекта

Приложение представляет собой простой интерфейс для ведения списка любимых песен. Каждая запись содержит название композиции, имя исполнителя и ссылку на музыкальный ресурс. Реализована возможность отмечать треки как избранные, фильтровать список и добавлять новые песни через небольшую форму.

# Особенности реализации

Написано на React с использованием функциональных компонентов.

Применяется useState для хранения списка песен, фильтра и данных формы.

Добавление песен реализовано через контролируемые поля <input>.

Статические данные используются как начальное состояние.

Стилизация CSS. 

Проект собирается стандартным Vite-шаблоном.

# Выводы по выполненной работе

Работа демонстрирует понимание ключевых основ React: компоненты, состояние, обработчики событий. Добавление песен показывает умение управлять пользовательским вводом.
